# 题目描述

请实现一个函数按照之字形打印二叉树，即第一行按照从左到右的顺序打印，第二层按照从右至左的顺序打印，第三行按照从左到右的顺序打印，其他行以此类推。

# 思路1：两个栈交错打印

和之前得一题“把二叉树打印成多行”类似，只不过那题是顺序打印，用的是两个队列。

## 代码

```java
import java.util.ArrayList;
import java.util.Stack;
/*
public class TreeNode {
    int val = 0;
    TreeNode left = null;
    TreeNode right = null;
    public TreeNode(int val) {
        this.val = val;
    }
}
*/
public class Solution {
    public ArrayList<ArrayList<Integer> > Print(TreeNode pRoot) {
        ArrayList<ArrayList<Integer>> list = new ArrayList<>();
        if (pRoot == null) return list;
        Stack<TreeNode> stack1 = new Stack<>();
        Stack<TreeNode> stack2 = new Stack<>();
        stack1.push(pRoot);
        while(!stack1.isEmpty() || !stack2.isEmpty()){
            ArrayList<Integer> a1 = new ArrayList<>();
            ArrayList<Integer> a2 = new ArrayList<>();
            while(!stack1.isEmpty()){
                TreeNode tmp = stack1.pop();
                a1.add(tmp.val);
                if (tmp.left != null){
                    stack2.push(tmp.left);
                }
                if (tmp.right != null){
                    stack2.push(tmp.right);
                }
            }
            if (a1.size() != 0) list.add(a1);
            while(!stack2.isEmpty()){
                TreeNode tmp = stack2.pop();
                a2.add(tmp.val);
                if (tmp.right != null){//这里和上面是相反的，先压右子节点
                    stack1.push(tmp.right);
                }
                if (tmp.left != null){
                    stack1.push(tmp.left);
                }
            }
            if (a2.size() != 0) list.add(a2);
        }
        return list;
    }
}
```

# 思路2：一个队列和一个栈

这题比较不方便的是返回值要求是个两层的ArrayList，而偶数层又需要翻转一下，存入ArayList不太方便，不然的话只用一个队列就可以。

要实现反转，借助栈的特性比较容易

## 代码

```java
import java.util.ArrayList;
import java.util.Stack;
public class Solution {
    public ArrayList<ArrayList<Integer> > Print(TreeNode pRoot) {
        ArrayList<ArrayList<Integer>> list = new ArrayList<>();//保存最终结果
        if (pRoot == null) return list;
        ArrayList<TreeNode> queue = new ArrayList<>();//遍历所有节点的队列
        Stack<Integer> stack = new Stack<>();//用来实现偶数层反序的栈
        queue.add(pRoot);
        boolean flag = true;//奇数层标志位
        while(queue.size() != 0){
            int size = queue.size();
            ArrayList<Integer> layer = new ArrayList<>();//保存每一层的结果
            if (flag){//如果在奇数层，用队列
                for (int i=0; i<size; i++){
                    TreeNode tmp = queue.remove(0);
                    layer.add(tmp.val);
                    if (tmp.left != null){
                        queue.add(tmp.left);
                    }
                    if (tmp.right != null){
                        queue.add(tmp.right);
                    }                
                    
                }
            }
            else{//如果在偶数层，用栈
                for (int i=0; i<size; i++){
                    TreeNode tmp = queue.remove(0);
                    stack.push(tmp.val);
                    if (tmp.left != null){
                        queue.add(tmp.left);
                    }
                    if (tmp.right != null){
                        queue.add(tmp.right);
                    }                                    
                    
                }
            }
            //将偶数层反序输出
            while(!flag && !stack.isEmpty()){
                layer.add(stack.pop());
            }
            flag = !flag;
            list.add(layer);
        }
        return list;
    }
}
```
